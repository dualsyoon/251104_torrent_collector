# 해결 방법 안내

## 문제 진단

```
ConnectionResetError: [WinError 10054] 현재 연결은 원격 호스트에 의해 강제로 끊겼습니다
```

이 오류는 **SSL 핸드셰이크 중에** 발생하고 있습니다.
즉, ISP(통신사)나 방화벽이 Python의 HTTPS 요청을 감지하고 차단하고 있습니다.

### 왜 브라우저는 되는데 Python은 안 될까?

- 브라우저: 다양한 SSL/TLS 설정, 브라우저 지문이 정상
- Python requests: 간단한 SSL 구현, 쉽게 감지됨
- ISP/방화벽이 Python 요청을 자동으로 차단

## ✅ 해결 방법

### 방법 1: VPN 사용 (가장 확실!)

**무료 VPN 추천:**

1. **ProtonVPN** (추천!)
   - 다운로드: https://protonvpn.com/download
   - 무료 플랜 사용 가능
   - 일본 서버 선택

2. **Windscribe**
   - 다운로드: https://windscribe.com/download
   - 월 10GB 무료
   - 일본/미국 서버 선택

3. **설정 방법:**
   ```
   1. VPN 설치 및 실행
   2. 일본 또는 미국 서버 연결
   3. VPN 연결 후 애플리케이션 실행
   4. python main.py
   ```

### 방법 2: 샘플 데이터로 테스트

실제 사이트 접속 없이 UI와 기능을 테스트:

```bash
# 1. 샘플 데이터 추가
python add_sample_data.py

# 2. 애플리케이션 실행
python main.py

# 3. 필터링, 검색 등 모든 기능 테스트 가능!
```

### 방법 3: 모바일 핫스팟 사용

일부 경우 모바일 데이터로는 접속 가능:

```
1. 스마트폰 핫스팟 켜기
2. PC를 핫스팟에 연결
3. 애플리케이션 실행
```

### 방법 4: 다른 PC/네트워크에서 시도

- 친구 집
- 카페 와이파이
- 회사 네트워크

## ❓ FAQ

**Q: VPN 없이는 불가능한가요?**
A: 한국 대부분의 ISP가 차단하므로 VPN이 거의 필수입니다.

**Q: 유료 VPN을 사야 하나요?**
A: 아니요! ProtonVPN 무료 플랜으로 충분합니다.

**Q: 샘플 데이터로만 사용 가능한가요?**
A: 네, UI와 모든 기능을 테스트할 수 있습니다!

**Q: 불법 아닌가요?**
A: VPN 사용은 합법이며, 개인 프라이버시 보호 도구입니다.
